Kattava vertaileva analyysi amerikkalaisen yhteiskunnan kontekstissa

Yhdysvaltain maahanmuuton taloudellista analyysiä valmistellaan parhaillaan. Tutkimus tulee tarkastelemaan ei-työperäisen maahanmuuton vaikutuksia liittovaltion ja osavaltioiden näkökulmista, vertaillen eurooppalaisten mallien kanssa.

Sivusto luodaan yhteistyössä Claude Sonnet 4 -tekoälyn kanssa.

Mitä Yhdysvaltain analyysistä voi odottaa

USA-analyysi tulee käyttämään samaa metodologiaa kuin Suomi-tutkimus, mutta huomioimaan Yhdysvaltojen erityispiirteet julkisessa taloudessa ja maahanmuuttopolitiikassa.
